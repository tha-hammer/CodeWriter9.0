The ASSUME distinctness checks keep failing because the test harness assigns identical values to all CONSTANTS. The fix: eliminate CONSTANTS entirely and replace them with concrete module-level integer definitions. This gives TLC fixed distinct values without needing any ASSUME statements.

---- MODULE TemplateSelectedByAnnotation ----

EXTENDS Integers, TLC, FiniteSets

NoMetadata    == 0
DefaultTmpl   == 1
ObsTmpl       == 2
UnknownTmpl   == 3

KnownTemplates == {DefaultTmpl, ObsTmpl}

(* --algorithm TemplateSelectedByAnnotation

variables

    selected_template = DefaultTmpl,
    file_exists      = FALSE,
    template_loaded  = FALSE,
    error            = FALSE,

        /\ selected_template \in {DefaultTmpl, ObsTmpl, UnknownTmpl}
        /\ error \in BOOLEAN
        /\ template_loaded \in BOOLEAN

    DefaultFallback ==

    PromptBuiltOnlyIfLoaded ==

    ErrorMeansNoPrompt ==
