# Context for gwt-0100: correction-agent-options

## Behavior
Given a correction agent is being constructed in `_run_correction_pass()`, when ClaudeAgentOptions are assembled, then allowed_tools is ["Read", "Grep", "Glob", "Bash", "Edit", "Write"]; max_turns is 20; model is "claude-sonnet-4-6"; system_prompt identifies the agent as a CW9 correction agent.

## Concrete Data Shapes

```
ClaudeAgentOptions == [
    allowed_tools: SEQUENCE(STRING),
    max_turns: NAT,
    model: STRING,
    system_prompt: STRING,
    output_format: STRING | NULL,
]

ExpectedTools == <<"Read", "Grep", "Glob", "Bash", "Edit", "Write">>
ExpectedMaxTurns == 20
ExpectedModel == "claude-sonnet-4-6"
```

## Key Invariants to Model

- **ExactToolSet**: allowed_tools = {"Read", "Grep", "Glob", "Bash", "Edit", "Write"} (no more, no less)
- **TurnLimit**: max_turns = 20
- **ModelFixed**: model = "claude-sonnet-4-6"
- **SystemPromptIdentity**: "correction" IN system_prompt (identifies agent role)
- **NoExtraTools**: "WebSearch" NOT IN allowed_tools AND "WebFetch" NOT IN allowed_tools

## Modeling Approach

- CONSTANTS: ToolSet (the 6 expected tools), MaxTurns (20), ModelName
- Variables: options (ClaudeAgentOptions)
- Actions: AssembleOptions
- Invariants: ExactToolSet, TurnLimit, ModelFixed, SystemPromptIdentity

## Test Interface (MANDATORY)

```python
import asyncio
from unittest.mock import AsyncMock, patch, MagicMock
from registry.cli import _run_correction_pass

captured_options = {}

async def mock_query(**kwargs):
    captured_options.update(kwargs)
    return MagicMock(result="corrected", cost_usd=0.02)

# Test: correction agent has correct options
with patch("registry.cli.query", side_effect=mock_query):
    result = asyncio.run(_run_correction_pass(
        pass_name="coverage",
        findings={"missing": ["test_foo"]},
        plan_path="/tmp/plan.md",
        cwd="/tmp/test",
        correction_prompt_file="prompts/cw9_plan_fix.md",
    ))
    opts = captured_options.get("options", captured_options)
    assert set(opts.get("allowed_tools", [])) == {"Read", "Grep", "Glob", "Bash", "Edit", "Write"}
    assert opts.get("max_turns") == 20
    assert opts.get("model") == "claude-sonnet-4-6"
    assert "correction" in opts.get("system_prompt", "").lower()
```

## Anti-Patterns (DO NOT USE)

- Do NOT give the correction agent TodoWrite or Agent tools -- it must not spawn sub-agents
- Do NOT set max_turns higher than 20 -- corrections should be bounded
- Do NOT use opus for corrections -- sonnet is sufficient and cheaper
- Do NOT omit the system_prompt -- the agent needs role identity to constrain its behavior
- Do NOT include "WebSearch" or "WebFetch" -- corrections are local filesystem operations only
