# Context for gwt-0049: scanner_typescript_nested_depth

## Behavior
Given a TypeScript source file with methods inside classes tracked by brace depth,
when scan_file is called,
then methods inside classes have class_name set to the enclosing class,
and functions outside classes have class_name=None.

TypeScript nesting: BRACE DEPTH COUNTING with class_stack.
