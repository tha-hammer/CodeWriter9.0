# Context for gwt-0047: scanner_python_nested_depth

## Behavior
Given a Python source file with nested functions inside classes at varying indent levels,
when scan_file is called,
then nested functions have class_name set to the enclosing class based on indent depth,
and top-level functions have class_name=None.
