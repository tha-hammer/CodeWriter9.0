# Context for gwt-0050: scanner_go_line_ranges

## Behavior
Given a Go source file with func declarations at known line positions,
when scan_file is called,
then each Skeleton.line_number equals the 1-indexed line of its func keyword.
Functions inside interface blocks are skipped.

## MANDATORY PlusCal Template

```tla
---- MODULE scanner_go_line_ranges ----

EXTENDS Integers, FiniteSets, TLC

Lines == << "Func", "Other", "InterfaceOpen", "Func", "InterfaceClose", "Func", "Other" >>
N == 7

(* --algorithm ScannerGoLineRanges

variables
    i            = 1,
    skeletons    = {},
    in_interface = FALSE;

define
    LineNumberCorrect ==
        \A s \in skeletons :
            /\ s.line_num >= 1
            /\ s.line_num <= N
            /\ Lines[s.line_num] = "Func"

    InterfaceExclusion ==
        \A s \in skeletons :
            \A k \in 1..N :
                (s.line_num = k) => Lines[k] = "Func"

    NoGaps ==
        i > N =>
            \A k \in 1..N :
                (Lines[k] = "Func" /\ ~InInterface(k)) =>
                    \E s \in skeletons : s.line_num = k

    \* Helper: is line k inside an interface block?
    InInterface(k) ==
        \E j \in 1..(k-1) :
            /\ Lines[j] = "InterfaceOpen"
            /\ ~\E m \in (j+1)..(k-1) : Lines[m] = "InterfaceClose"

    CursorBounded == i <= N + 1
end define;

process scanner = "scanner"
begin
    ScanLoop:
        while i <= N do
            ProcessLine:
                if Lines[i] = "InterfaceOpen" then
                    in_interface := TRUE;
                elsif Lines[i] = "InterfaceClose" then
                    in_interface := FALSE;
                elsif Lines[i] = "Func" /\ ~in_interface then
                    skeletons := skeletons \cup {[line_num |-> i]};
                end if;
            Advance:
                i := i + 1;
        end while;
    Finish:
        skip;
end process;

end algorithm; *)

====
```

RULES: Output ONLY the module once. No CONSTANTS. No RECURSIVE. No CHOOSE. No THEOREM after ====.

## Test Interface (MANDATORY — use these exact imports and patterns)

```python
from pathlib import Path
from registry.scanner_go import scan_file, scan_directory
from registry.crawl_types import Skeleton, SkeletonParam

# scan_file takes a Path, NOT a string
result: list[Skeleton] = scan_file(Path("/tmp/test.go"))

# Skeleton is a dataclass — use ATTRIBUTE access, NOT subscript
assert result[0].function_name == "hello"    # CORRECT
assert result[0].line_number == 1            # CORRECT
assert result[0].class_name == "MyClass"     # CORRECT (or None)
assert result[0].visibility == "public"      # CORRECT
assert result[0].is_async == False           # CORRECT
assert result[0].return_type == "str"        # CORRECT (or None)
```

Anti-Patterns (DO NOT USE — these cause test failures)

- `skeleton["function_name"]` ← WRONG. Skeleton is a dataclass, not a dict
- `skeleton.func_name` ← WRONG. The field is `function_name`
- `skeleton.def_indent` ← DOES NOT EXIST on Skeleton
- `skeleton.inside_class` ← DOES NOT EXIST. Use `skeleton.class_name is not None`
- `scan_file("path/to/file")` ← WRONG. Must be `scan_file(Path("path/to/file"))`
- `scan_file(str)` ← WRONG type. Signature is `scan_file(path: Path) -> list[Skeleton]`
