# Context for gwt-0051: scanner_go_nested_depth

## Behavior
Given a Go source file with receiver methods on struct types,
when scan_file is called,
then receiver methods have class_name set to the receiver type,
and package-level functions have class_name=None.

Go nesting: RECEIVER TYPE IN SIGNATURE (no stack needed).
Visibility: capitalized name = "public", lowercase = "private".

## MANDATORY PlusCal Template

```tla
---- MODULE scanner_go_nested_depth ----

EXTENDS Integers, FiniteSets, TLC

\* Event stream: each event is a func declaration
\* has_receiver: TRUE if method with receiver, FALSE if package func
\* recv_type: receiver type name or "None"
\* capitalized: TRUE if func name starts with uppercase
Events == <<
    [has_recv |-> FALSE, recv_type |-> "None",  name |-> "Hello",    cap |-> TRUE],
    [has_recv |-> TRUE,  recv_type |-> "Svc",   name |-> "GetUser",  cap |-> TRUE],
    [has_recv |-> TRUE,  recv_type |-> "Point", name |-> "distance", cap |-> FALSE],
    [has_recv |-> FALSE, recv_type |-> "None",  name |-> "helper",   cap |-> FALSE]
>>
N == 4

(* --algorithm ScannerGoNestedDepth

variables
    cursor    = 1,
    skeletons = {};

define
    ReceiverResolution ==
        \A s \in skeletons :
            \/ (s.class_name = "None" /\ s.has_recv = FALSE)
            \/ (s.class_name # "None" /\ s.has_recv = TRUE)

    VisibilityCorrect ==
        \A s \in skeletons :
            \/ (s.visibility = "public"  /\ s.cap = TRUE)
            \/ (s.visibility = "private" /\ s.cap = FALSE)

    AllRecorded ==
        cursor > N =>
            \A k \in 1..N :
                \E s \in skeletons : s.func_name = Events[k].name
end define;

process scanner = "scanner"
begin
    ScanLoop:
        while cursor <= N do
            ProcessEvent:
                skeletons := skeletons \cup
                    {[func_name  |-> Events[cursor].name,
                      class_name |-> Events[cursor].recv_type,
                      has_recv   |-> Events[cursor].has_recv,
                      cap        |-> Events[cursor].cap,
                      visibility |-> IF Events[cursor].cap THEN "public" ELSE "private"]};
            Advance:
                cursor := cursor + 1;
        end while;
    Finish:
        skip;
end process;

end algorithm; *)

====
```

RULES: Output ONLY the module once. No CONSTANTS. No RECURSIVE. No CHOOSE. No THEOREM after ====.

## Test Interface (MANDATORY — use these exact imports and patterns)

```python
from pathlib import Path
from registry.scanner_go import scan_file, scan_directory
from registry.crawl_types import Skeleton, SkeletonParam

# scan_file takes a Path, NOT a string
result: list[Skeleton] = scan_file(Path("/tmp/test.go"))

# Skeleton is a dataclass — use ATTRIBUTE access, NOT subscript
assert result[0].function_name == "hello"    # CORRECT
assert result[0].line_number == 1            # CORRECT
assert result[0].class_name == "MyClass"     # CORRECT (or None)
assert result[0].visibility == "public"      # CORRECT
assert result[0].is_async == False           # CORRECT
assert result[0].return_type == "str"        # CORRECT (or None)
```

Anti-Patterns (DO NOT USE — these cause test failures)

- `skeleton["function_name"]` ← WRONG. Skeleton is a dataclass, not a dict
- `skeleton.func_name` ← WRONG. The field is `function_name`
- `skeleton.def_indent` ← DOES NOT EXIST on Skeleton
- `skeleton.inside_class` ← DOES NOT EXIST. Use `skeleton.class_name is not None`
- `scan_file("path/to/file")` ← WRONG. Must be `scan_file(Path("path/to/file"))`
- `scan_file(str)` ← WRONG type. Signature is `scan_file(path: Path) -> list[Skeleton]`
