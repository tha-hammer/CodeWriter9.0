# Context for gwt-0051: scanner_go_nested_depth

## Behavior
Given a Go source file with receiver methods on struct types,
when scan_file is called,
then receiver methods have class_name set to the receiver type,
and package-level functions have class_name=None.

Go nesting: RECEIVER TYPE IN SIGNATURE (no stack needed).
Visibility: capitalized name = "public", lowercase = "private".
