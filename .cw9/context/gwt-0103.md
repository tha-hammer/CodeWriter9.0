# Context for gwt-0103: correction-agent-input-from-manifest

## Behavior
Given a correction agent is being assembled and `_review_manifest.json` has the latest failed attempt for a pass, when `_run_correction_pass()` resolves its input, then the report_path from the manifest entry is resolved to an absolute path; the agent is instructed to read the on-disk report; if the report file is missing, an error is raised and no agent is spawned.

## Concrete Data Shapes

```
ManifestEntry == [
    pass_name: STRING,
    attempt: NAT,
    verdict: STRING,
    report_path: STRING,
    session_id: STRING,
    cost: REAL,
]

CorrectionInput == [
    report_path: STRING,   \* absolute, resolved from manifest
    findings: DICT,        \* extracted from the report
]

ResolveResult == [
    success: BOOLEAN,
    absolute_path: STRING | NULL,
    error: STRING | NULL,
]
```

## Key Invariants to Model

- **PathResolved**: report_path from manifest is resolved to an absolute path before use
- **FileExists**: if resolved path does not exist on disk, error is raised
- **NoAgentOnMissing**: if report file is missing, _run_correction_pass raises error, no query() call made
- **LatestAttempt**: the manifest entry used is the most recent failed attempt for the given pass_name
- **AgentReadsReport**: the correction agent's prompt includes the resolved report_path for reading

## Modeling Approach

- CONSTANTS: PassNames, FileExists (BOOLEAN)
- Variables: manifest (SEQUENCE of ManifestEntry), resolved_path (STRING | NULL), agent_spawned (BOOLEAN), error_raised (BOOLEAN)
- Actions: ResolveFromManifest(pass_name), SpawnAgent, RaiseError
- Invariants: NoAgentOnMissing, PathResolved, LatestAttempt

## Test Interface (MANDATORY)

```python
import asyncio, json
from unittest.mock import AsyncMock, patch, MagicMock
from pathlib import Path
from registry.cli import _run_correction_pass

# Setup: create manifest with a failed attempt
manifest_dir = Path("/tmp/test_correction")
manifest_dir.mkdir(parents=True, exist_ok=True)
manifest_path = manifest_dir / "_review_manifest.json"

# Test: report exists -> agent spawned with correct path
report_path = manifest_dir / "coverage_report.md"
report_path.write_text("FAIL: missing assertions for test_bar")

manifest_data = {
    "entries": [
        {
            "pass_name": "coverage",
            "attempt": 1,
            "verdict": "fail",
            "report_path": str(report_path),
            "session_id": "sess-001",
            "cost": 0.05,
        }
    ]
}
manifest_path.write_text(json.dumps(manifest_data))

captured_prompt = {}

async def mock_query(**kwargs):
    captured_prompt.update(kwargs)
    return MagicMock(result="corrected", cost_usd=0.02)

with patch("registry.cli.query", side_effect=mock_query):
    result = asyncio.run(_run_correction_pass(
        pass_name="coverage",
        findings={"missing": ["test_bar"]},
        plan_path="/tmp/plan.md",
        cwd=str(manifest_dir),
        correction_prompt_file="prompts/cw9_plan_fix.md",
    ))
    assert str(report_path) in captured_prompt.get("prompt", "")

# Test: report file missing -> error raised, no agent
missing_report = manifest_dir / "nonexistent_report.md"
manifest_data["entries"][0]["report_path"] = str(missing_report)
manifest_path.write_text(json.dumps(manifest_data))

import pytest
with pytest.raises(Exception):
    asyncio.run(_run_correction_pass(
        pass_name="coverage",
        findings={"missing": ["test_bar"]},
        plan_path="/tmp/plan.md",
        cwd=str(manifest_dir),
        correction_prompt_file="prompts/cw9_plan_fix.md",
    ))
```

## Anti-Patterns (DO NOT USE)

- Do NOT embed the report content in the manifest -- store only the path, let the agent read the file
- Do NOT silently skip missing reports -- raise an explicit error so the user knows the report was lost
- Do NOT use relative paths -- always resolve to absolute before passing to the correction agent
- Do NOT use an older attempt when a newer failed attempt exists for the same pass
- Do NOT pass the review session_id to the correction agent -- it starts fresh (see gwt-0099)
