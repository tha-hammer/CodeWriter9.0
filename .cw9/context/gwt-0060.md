# Context for gwt-0060: go_profile_condition_compile

## Behavior
Given TLA+ expressions, when GoProfile.compile_condition is called,
a CompiledExpression is returned. Go is the ONLY profile that emits helper_defs
for quantifiers (allSatisfy/anySatisfy helper functions).
