# Context for gwt-0099: correction-agent-fresh-context

## Behavior
Given a correction agent is being spawned for a failed review pass, when `_run_correction_pass()` constructs the `query()` call, then resume is NOT set (no session continuation); no session ID from the review pass is passed; the agent gets a clean context containing only the findings from the failed pass and the report path.

## Concrete Data Shapes

```
QueryArgs == [
    prompt: STRING,
    resume: BOOLEAN,       \* must be FALSE
    session_id: STRING,    \* must be absent/None
    options: AgentOptions,
]

CorrectionInput == [
    pass_name: STRING,
    findings: DICT,        \* extracted findings from the failed review
    report_path: STRING,   \* absolute path to the review report on disk
]
```

## Key Invariants to Model

- **NoResume**: query() call has resume = FALSE (or resume not set)
- **NoSessionCarryover**: query() call does NOT include the session_id from the review pass
- **FindingsPresent**: the prompt includes the findings dict serialized into the message
- **ReportPathPresent**: the prompt includes the absolute report_path so the agent can read it
- **CleanContext**: no prior conversation history is injected -- the correction agent starts fresh

## Modeling Approach

- CONSTANTS: PassNames (set of review pass names)
- Variables: query_args (QueryArgs), correction_input (CorrectionInput)
- Actions: ConstructCorrectionQuery(pass_name, findings, report_path)
- Invariants: NoResume, NoSessionCarryover, FindingsPresent, ReportPathPresent

## Test Interface (MANDATORY)

```python
import asyncio
from unittest.mock import AsyncMock, patch, MagicMock
from registry.cli import _run_correction_pass

# Mock the SDK query() to capture the arguments it receives
captured_args = {}

async def mock_query(**kwargs):
    captured_args.update(kwargs)
    return MagicMock(result="corrected", cost_usd=0.02)

# Test: correction pass does not set resume or session_id
with patch("registry.cli.query", side_effect=mock_query):
    result = asyncio.run(_run_correction_pass(
        pass_name="coverage",
        findings={"missing": ["test_foo"]},
        plan_path="/tmp/plan.md",
        cwd="/tmp/test",
        correction_prompt_file="prompts/cw9_plan_fix.md",
    ))
    assert captured_args.get("resume") is not True
    assert captured_args.get("session_id") is None or "session_id" not in captured_args
    assert "missing" in str(captured_args.get("prompt", ""))
```

## Anti-Patterns (DO NOT USE)

- Do NOT reuse the review session -- correction must start from scratch to avoid poisoned context
- Do NOT pass the full review conversation transcript -- only pass structured findings and report path
- Do NOT model the correction agent's internal execution -- only model the query() construction
- Do NOT conflate "no session_id" with "empty string" -- session_id should be absent, not ""
