# Context for gwt-0080: plan-review-phase-filtering

## Behavior
Given phase is one of 'pre', 'post', or 'all', when `_orchestrate_reviews()` selects which passes to run, then phase='pre' runs artifacts+coverage+interaction+abstraction_gap but NOT imports; phase='post' runs ONLY imports; phase='all' runs everything; external mode never includes interaction regardless of phase.

## Concrete Data Shapes

```
Phase == {"pre", "post", "all"}
Mode == {"self", "external"}
PassName == {"artifacts", "coverage", "interaction", "abstraction_gap", "imports"}

FilterResult == [executed_passes: SUBSET(PassName)]
```

## Key Invariants to Model

- **PreSkipsImports**: phase="pre" => "imports" NOT IN executed_passes
- **PostOnlyImports**: phase="post" => executed_passes = {"imports"} (if imports key exists)
- **AllRunsEverything**: phase="all" AND mode="self" AND artifacts passes => executed_passes = all 5
- **ExternalNoInteraction**: mode="external" => "interaction" NOT IN executed_passes (regardless of phase)
- **AllExternalNoInteraction**: phase="all" AND mode="external" AND artifacts passes => executed_passes = {"artifacts", "coverage", "abstraction_gap", "imports"}
- **PreExternalNoInteraction**: phase="pre" AND mode="external" => executed_passes subset of {"artifacts", "coverage", "abstraction_gap"}

## Modeling Approach

- CONSTANTS: Phase (element of Phase set), Mode (element of Mode set), ArtifactsVerdict (pass/fail)
- Variables: executed (set of PassName), current_phase_step (enum)
- Actions: ExecutePass(name), SkipPass(name)
- Invariants: PreSkipsImports, PostOnlyImports, ExternalNoInteraction

## Test Interface (MANDATORY)

```python
import asyncio
from registry.cli import _orchestrate_reviews

# Test phase="pre" skips imports
result = asyncio.run(_orchestrate_reviews("self", "plan.md", "/tmp/test", "pre", True))
assert "imports" not in result["results"]

# Test phase="post" only runs imports
result = asyncio.run(_orchestrate_reviews("self", "plan.md", "/tmp/test", "post", True))
assert list(result["results"].keys()) == ["imports"]

# Test external mode never includes interaction
result = asyncio.run(_orchestrate_reviews("external", "plan.md", "/tmp/test", "all", True))
assert "interaction" not in result["results"]
```

## Anti-Patterns (DO NOT USE)

- Do NOT model the contents of each pass — just model whether it executes or not
- Do NOT model ordering here — that's gwt-0077's concern
- Keep phase and mode as CONSTANTS not variables (they don't change mid-execution)
